## üîÅ Core User Flow

### 1. Experience Setup (Brand / Organizer)

- Create a new event or campaign
- Choose or define the visual theme (prompt, style, references)
- Configure settings (input type, branding, output format)
- Receive a sharable event link or embeddable widget

### 2. User Interaction (End User / Guest)

- Visit the link (mobile-first experience)
- Take or upload a selfie
- See their image transformed via AI in under 1 minute
- Download/share on social or save to gallery

### 3. Analytics & Upsell (Post-event Review)

Event creators receive access to:

- Views, submissions, shares, engagement stats
- Top prompts/themes
- Visual gallery of results

**Optional**: Clementine team provides tailored performance summaries and upsell opportunities (e.g. for recurring events, seasonal campaigns, influencer collabs)
